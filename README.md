# Edição de Imagens com IA

Projeto acadêmico para desenvolvimento de um sistema de edição localizada de imagens guiada por comandos curtos em português brasileiro.

## Objetivo

Receber uma imagem e uma instrução em português, identificar o objeto mencionado e realizar uma edição apenas na região desejada.

## Operações previstas

- Alteração de brilho
- Alteração de cor
- Remoção simples de objetos

## Tecnologias

- Python
- Google Colab
- OpenCV
- PyTorch
- Modelos de difusão

## Estrutura do projeto

- `data/`: imagens e dados
- `docs/`: documentação
- `notebooks/`: notebooks do Google Colab
- `src/`: código-fonte
- `outputs/`: resultados gerados
- `experiments/`: testes e experimentos
